#Day 2 Notes
Programming languages have two types of techniques to pass arguments:
Pass/Call by value

{% highlight java %}
Class Cat{
	String name;
	
	Public Cat (String name) { 
		this.name = name; } }
		
Static void foo(Cat d) {
	d.name = "Baloo"; // c
	d = new Cat("Lilo"); } //d
	
Static main(String args[]){
	Cat aCat = new Cat("Foo"); //a
	Cat.Foo(aCat); } //b
